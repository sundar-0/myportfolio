1. Database Management System

1.1 Introduction to Data, Database and DBMS
Data means raw facts. Database is an organized collection of data. DBMS is software used to store, manage, update and retrieve data easily.
1.2 Keys in DBMS
Primary key uniquely identifies each record. Candidate key can become primary key. Alternate key is candidate key not selected as primary key.
1.3 Advantages of DBMS
DBMS reduces data redundancy, improves security, supports data sharing, backup, recovery and maintains data consistency.
1.4 DDL and DML
DDL defines database structure like CREATE, ALTER, DROP. DML manipulates data like INSERT, UPDATE, DELETE, SELECT.

2. Data Communication & Networking

2.1 Communication System
Communication system transfers information from sender to receiver through a medium using rules called protocol.
2.2 Communication Modes
Simplex sends data one way only. Half duplex sends both ways but one at a time. Full duplex sends both ways at the same time.
2.3 LAN and WAN
LAN covers small area like school or office. WAN covers large area like city, country or world.
2.4 OSI Model
OSI model has 7 layers: Physical, Data Link, Network, Transport, Session, Presentation and Application.

3. Web Technology II

3.1 Introduction
Web Technology II means advanced web development. यसमा website कसरी interactive, dynamic र database connected बनाउने भन्ने पढिन्छ। It includes JavaScript, PHP, form validation, server-side scripting and database connectivity.
3.2 Client Side & Server Side Scripting
Client-side scripting browser मा run हुन्छ, like JavaScript. Server-side scripting server मा run हुन्छ, like PHP. Client side UI interaction को लागि हो भने server side database, login, processing को लागि हो।
3.3 Internet Technology
Internet Technology includes browser, server, URL, domain, hosting, HTTP/HTTPS and web communication system.
3.4 JavaScript in HTML
JavaScript HTML मा interactive behavior add गर्न प्रयोग हुन्छ। It can be written inside script tag or linked as external JS file.
3.5 JavaScript Fundamentals
JavaScript fundamentals include variables, data types, operators, functions, loops, conditions and events.
3.6 Data Types
Data type means type of value stored in variable. JavaScript data types are String, Number, Boolean, Null, Undefined, Object and Array.
3.7 Variables and Operators
Variables store data. Operators perform calculation and comparison. Examples: +, -, *, /, ==, ===, >, <.
3.8 Control Structures
Control structures control program flow. Examples: if, else, switch, for loop, while loop and do while loop.
3.9 Object Based JS & Event Handling
Object based JavaScript uses objects like document, form, image and window. Event handling means responding to user actions like click, submit, keypress and mouseover.

4. Programming in C

4.1 Review of C Programming
C is a structured programming language. It supports variables, data types, operators, loops, functions, arrays and pointers.
4.2 Functions
Function is a block of code used to perform a specific task. It helps to reduce repetition and make program modular.
4.3 Structures and Unions
Structure stores different data types under one name. Union also stores different data types but shares same memory location.
4.4 Pointers
Pointer is a variable that stores address of another variable. It is useful in memory handling, arrays, functions and dynamic allocation.
4.5 File Handling
File handling is used to store data permanently in files. Common operations are open, read, write, append and close.

5. Object Oriented Programming

5.1 Class and Object
Class is a blueprint. Object is an instance of class. Example: Student is class, Ram is object.
5.2 Polymorphism
Polymorphism means one name many forms. Same function can perform different tasks.
5.3 Inheritance
Inheritance allows one class to use properties and methods of another class. It supports code reusability.

6. Software Process Model

6.1 SDLC
SDLC means Software Development Life Cycle. It includes planning, analysis, design, coding, testing, deployment and maintenance.
6.2 Waterfall Model
Waterfall model is a sequential software development model where each phase must be completed before next phase starts.
6.3 Agile Model
Agile model is flexible and iterative. Software is developed in small parts with continuous feedback.

7. Recent Trends

7.1 Artificial Intelligence
AI is a branch of computer science that makes machines capable of learning, reasoning and decision making.
7.2 Cloud Computing
Cloud computing provides computing services like storage, database and software through internet.
7.3 IoT
Internet of Things connects physical devices to internet so they can collect and exchange data.
